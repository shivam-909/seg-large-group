## Installation Instructions
1. Clone the repository.
2. ```cd backend```
3. ```npm install```
4. ```npm run build```
5. ```npm run dev```
6. ```cd ..```
7. ```cd frontend```
8. ```npm install```
9. ```npm run build```
10. ```npm run start```

## Testing Instructions
Run frontend tests:
1. ```cd frontend```
2. ```npm run test:coverage``` <br>

Run backend tests:
1. ```cd backend```
2. ```npx jest --coverage```
